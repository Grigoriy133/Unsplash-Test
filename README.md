#  Тестовое задание 
___
Приложение загружает случайные картинки с сервиса Unsplash, позволяет сохранить понравившиеся изображения на отдельную вкладку 
Показывает описание изображения
# Стек
___
 * UIKit
 * AutoLayout
 * MVC
 * UserDefaults
 * Alamofire
 * KingFisher
